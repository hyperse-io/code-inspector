# About Code Inspector

**Code Inspector** is developed by [Hyperse](https://github.com/hyperse-io) as
part of a broader developer-experience toolkit alongside projects such as
[Hyperse HPS](https://hyperse-io.github.io/hps/).

The documentation you are reading is built with [Nextra 4](https://nextra.site)
on [Next.js](https://nextjs.org) and deployed to GitHub Pages.
